Understanding Tokens

In English, a token is roughly three to four characters, or about three quarters of a word on average.

Why Tokens Matter

When you send a message to an AI model, it does not process words. It processes tokens. Understanding how tokenisation works is one of the most practical skills you can build as a developer working with AI.

Tokens are important for:

  • Pricing: AI providers charge per token, not per word
  • Context Windows: Models have limits on how many tokens they can process
  • Efficiency: Optimizing token usage saves money and improves performance

How We Approximate Tokens

The exact number depends on the tokeniser each model uses internally. However, we can build a practical approximation:

  1. Take a string of English text as input
  2. Remove extra whitespace from the beginning and end
  3. Split the text into an array of words
  4. Remove any empty items from the array
  5. Count the words and multiply by 0.75 to get the approximate token count
  6. Round up to the nearest whole number so you never undercount
